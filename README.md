# Para a Dudinha 💗

Site de página única, **100% autocontido** (fotos e música embutidas no HTML) — não precisa de build, servidor nem dependências.

## Como publicar no GitHub Pages
1. Crie um repositório novo no GitHub (ex.: `para-dudinha`).
2. Envie estes arquivos pelo **Add file → Upload files**: `index.html`, `.nojekyll` e `README.md`.
3. Em **Settings → Pages**: *Source* = `Deploy from a branch`, *Branch* = `main`, pasta `/ (root)` → **Save**.
4. Em ~1 minuto o site estará no ar em: `https://SEU_USUARIO.github.io/NOME_DO_REPO/`

> **Privacidade:** no plano gratuito o GitHub Pages exige repositório **público** — qualquer pessoa com o link consegue ver. Se quiser discrição, use um nome de repositório difícil de adivinhar.
